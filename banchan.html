할머니 주방이란?

할머니들이 직접 만들고 배달 혹은 고객이 픽업할 수 있는 동네 반찬 거래 어플리케이션입니다.
동네에서 쉽게 주고 받는 할머니들 반찬 거래! 이제는 가게에서 사지마세요!
    ✔️ 편하게 할머니들 산책겸 배달로 받아보세요!
    ✔️ 직접 할머니댁 앞으로 가져가는 선택지도 있어요!

👵🏻 사용 방법 [할머니]

  1. 앱을 다운로드한다. (위치 기능 허용해야함)
  2. 간단한 전화번호 본인인증 후 가입한다.
  3. 반찬 이름을 리스트에서 선택한다.
  4. 직접 만든 사진을 등록한다.
  5. 알고리즘을 통해 직접 만든 반찬인 것 확인한다.
  6. 상세 설명과 원하는 거래 방식 그리고 가격을 등록한다.

👩🏻 사용 방법 [손자손녀]

  1. 앱을 다운로드한다. (위치 기능 허용해야함)
  2. 간단한 전화번호 본인인증 후 가입한다.
  3. 동네 지역 지도를 클릭하여 주변 할머니를 확인한다.
  4. 원하는 할머니 선택 후, 반찬을 선택하고 구입을 누른다.
  5. 거래방식은 두 가지 중 선택한다.
